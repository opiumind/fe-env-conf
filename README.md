# fe-env-conf
Автоматичсекая настройка среды fe разработчика НГС

Данный инструмент создан, чтобы не настраивать созданную или пересозданную среду вручную.

Алгоритм действий:
1. Занесите необходимые данные в конфиг fe-env-conf-config.sh (необязательно)
2. Запустите скрипт fe-env-conf-start.sh у себя на локальной машине
3. Если все необходимые данные в конфиге, расслабьтесь и отдохните
